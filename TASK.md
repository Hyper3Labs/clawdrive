ClawDrive - a google drive but for the 2026 world of AI Agents
- it is google drive but for agents like openclaw (research this if you don't know what these mean)
- we will have local version and hosted one. the local version should be possible to host on your own and make it so the agents can communicate with each other and share files
- use something like https://github.com/volcengine/OpenViking (see first if it makes sense at all to use it) which has different levels and combine it with agentic search
- use gemini embedding 2 (the lastest one that just came out in March 2026)
- for the ui one idea is a  3d plot of the embeddings like https://tom-doerr.github.io/repo_posts/map.html (see the 3d map of the embeddings and how it shows snippets) but that might not be too useful for humans to look at but could be useful for agents
- for humans you can organize things into hierarchies/taxonomoy, one main topic/sub-topic having no more than 8 things inside of it. this is analogous to having folders and subfolders...
- we can have a tab that can switch between agent view and human view

- keyboard shortcut to open a kind of spotlight search, where the user enters what they want, which triggers an agentic search across the files
