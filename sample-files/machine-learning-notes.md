# Machine Learning Fundamentals

## Supervised Learning
Training with labeled data. Classification (discrete labels) and regression (continuous values). Loss functions: cross-entropy, MSE.

## Neural Network Architecture
Layers: input, hidden, output. Activation functions: ReLU, sigmoid, softmax. Backpropagation for gradient computation.

## Transformers
Self-attention mechanism allows modeling long-range dependencies. Multi-head attention, positional encoding, layer normalization.
