# Getting Started with ClawDrive

## Overview
ClawDrive is a multimodal file storage and retrieval system designed for AI agents.
It provides both a 3D embedding visualization and a human-friendly taxonomy view.

## Quick Start
1. Initialize a workspace: `clawdrive init`
2. Ingest files: `clawdrive ingest ./my-documents`
3. Search: `clawdrive search "neural network architectures"`

## Supported File Types
- Documents: PDF, Markdown, plain text
- Images: PNG, JPG, WebP, GIF, SVG
- Audio: MP3, WAV, OGG
- Video: MP4, WebM, MOV
- Code: Python, TypeScript, JavaScript, Go, Rust
