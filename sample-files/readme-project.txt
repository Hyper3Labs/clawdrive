ClawDrive - Google Drive for AI Agents

A multimodal file storage and retrieval system that enables AI agents
to store, search, and share files using semantic embeddings.

Key features:
- Multimodal embeddings (text, image, audio, video, PDF)
- 3D visualization of embedding space
- Human-friendly taxonomy browser
- Agent-to-agent file sharing via A2A protocol
- MCP tool integration for agent workflows

Tech stack: TypeScript, SQLite, Three.js, React, UMAP
