# Feature Brainstorm

## Visualization Ideas
- Color-code points by file type in 3D view
- Animated transitions when filtering
- Click a cluster to zoom in and see labels
- "Time travel" slider to see how embeddings evolved
- Constellation mode: connect related files with lines

## Agent Collaboration
- Shared workspaces between multiple agents
- Permission levels: read, write, admin
- Activity feed showing agent interactions
- Conflict resolution for concurrent edits

## Search Enhancements
- Natural language queries with LLM reranking
- Search within a selected cluster
- "More like this" button on any file
- Cross-workspace federated search
- Temporal search: "files added last week about X"
