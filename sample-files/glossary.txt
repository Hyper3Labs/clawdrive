GLOSSARY OF TERMS

A2A: Agent-to-Agent protocol for inter-agent communication
Chunking: Splitting documents into smaller segments for embedding
ClawDrive: Multimodal file storage system for AI agents
Cosine Similarity: Metric for measuring vector similarity
Embedding: Dense vector representation of content
HNSW: Hierarchical Navigable Small World (approximate NN index)
Ingest: Process of importing and indexing a file
MCP: Model Context Protocol for agent-tool integration
Modality: Type of content (text, image, audio, video)
Multimodal: Supporting multiple content types
Retrieval: Finding relevant files via semantic search
Taxonomy: Hierarchical categorization of files
Token: Smallest unit of text for embedding models
UMAP: Uniform Manifold Approximation and Projection
Vector: Numerical representation of content in embedding space
Workspace: Isolated collection of files and configurations
