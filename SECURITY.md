# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in ClawDrive, **please report it responsibly**.

**Do not open a public GitHub issue for security vulnerabilities.**

Instead, email **security@hyper3labs.com** with:

- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

We will:
- Acknowledge receipt within **48 hours**
- Aim to provide a fix within **7 days** for critical issues

## Supported Versions

| Version | Supported |
|---|---|
| 0.1.x (latest) | Yes |

## Scope

This policy covers the ClawDrive codebase. Third-party dependency vulnerabilities should be reported to their respective maintainers.
